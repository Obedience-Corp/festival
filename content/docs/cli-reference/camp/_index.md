# camp CLI Reference

`camp` manages campaigns — multi-project AI workspaces with structured directory layouts for projects, documentation, and planning.

## Key Commands

| Command | Description |
|---------|-------------|
| `camp init` | Create a new campaign workspace |
| `camp project add <url>` | Add a project repository as a submodule |
| `camp commit -m "msg"` | Commit with campaign tracking and auto-sync |
| `camp doctor` | Health check the campaign workspace |
| `camp sync` | Sync all project submodules |
| `camp switch` | Switch between registered campaigns |
