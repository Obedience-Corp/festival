## fest show dungeon

List festivals in dungeon/ directory

### Synopsis

List festivals in dungeon/ directory.

Optionally specify a substatus: completed, archived, someday.
Without a substatus, lists all dungeon festivals.

```
fest show dungeon [substatus] [flags]
```
