Festival Methodology is a goal-oriented project management system designed for AI-assisted development workflows. It provides a structured hierarchy for planning, executing, and tracking complex software projects.

## Core Principles

- **Goal-oriented** — Every festival has a clear, measurable objective
- **AI-native** — Designed for human-AI collaboration from the ground up
- **Hierarchical** — Five levels of granularity from campaign to task
- **Lifecycle-aware** — Each element has defined status transitions

## Hierarchy

```
Campaign → Festival → Phase → Sequence → Task
```

## Who Is This For?

- **Solo developers** using AI agents for autonomous development
- **Teams** coordinating complex multi-phase projects
- **AI agents** operating in long-running autonomous sessions
