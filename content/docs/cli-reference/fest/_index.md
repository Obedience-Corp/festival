# fest CLI Reference

`fest` manages the Festival Methodology — a goal-oriented project management system designed for AI agent development workflows.

## Key Commands

| Command | Description |
|---------|-------------|
| `fest intro` | Getting started guide for new sessions |
| `fest next` | Get the next task with full inline context |
| `fest status` | View festival progress overview |
| `fest create festival` | Create a new festival |
| `fest commit -m "msg"` | Commit with festival tracking metadata |
| `fest understand [topic]` | Learn methodology concepts on demand |
| `fest validate` | Validate festival structure |
| `fest context` | Get context for current location |
