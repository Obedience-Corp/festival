A festival is a goal-oriented project plan. It defines what needs to be accomplished and breaks the work into phases, sequences, and tasks.

## Festival Lifecycle

Festivals move through defined status directories:

- **planning/** — Being designed and scoped
- **ready/** — Fully planned, awaiting execution
- **active/** — Currently being executed
- **dungeon/completed/** — Successfully finished
- **dungeon/archived/** — Shelved but preserved
- **dungeon/someday/** — Deprioritized for later

## Creating a Festival

```bash
fest create festival my-festival-name
```
