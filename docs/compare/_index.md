# Compare Festival

Festival is a persistent multi-repo workspace for AI coding agents. These pages help you place it next to adjacent tools without forcing a false either-or.

Festival is not a hosted orchestrator, prompt pack, or generic task board. It is the local-first mission layer that keeps repos, docs, plans, next tasks, done criteria, and commits tied together on disk.
