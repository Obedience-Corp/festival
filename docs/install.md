## Linux

Choose your distribution's package format:

| Distribution | Command |
|-------------|---------|
| Debian / Ubuntu | `sudo dpkg -i festival_*_amd64.deb` |
| Fedora / RHEL | `sudo rpm -i festival-*.x86_64.rpm` |
| Arch Linux | `yay -S festival-bin` |
| Alpine | `sudo apk add --allow-untrusted festival_*.apk` |

[Download Linux Packages](https://github.com/Obedience-Corp/festival/releases/latest){ .md-button .md-button--primary }

</div>

<div id="install-windows" class="install-platform" style="display:none" markdown>

## Windows

Install with Scoop:

```powershell
scoop bucket add obey https://github.com/Obedience-Corp/scoop-bucket
scoop install festival
```

Or download the `.zip` directly:

[Download for Windows](https://github.com/Obedience-Corp/festival/releases/latest){ .md-button .md-button--primary }

</div>
