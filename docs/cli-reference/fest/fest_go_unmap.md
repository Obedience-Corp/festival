## fest go unmap

Remove a navigation shortcut

### Synopsis

Remove a previously created navigation shortcut.

```
fest go unmap <name> [flags]
```

### Examples

```
  fest go unmap n     # Remove shortcut 'n'
  fest go unmap api   # Remove shortcut 'api'
```

      --json   output in JSON format
